# Ярлыкон

Персональный шортенер ссылок. Git как источник правды, Cloudflare Workers как тонкая логика, Telegram/VK/MAX Mini App как фронт.

Каждая короткая ссылка — это HTML-файл с зашитым OG-снапшотом назначения, Яндекс.Метрикой и UA-чувствительным редиректом. Получатель видит корректное превью, человек попадает на цель, ты видишь статистику.

## Зачем

- **Свой домен**, свой контроль, свои данные.
- **Аудит-журнал бесплатно** — каждая ссылка это коммит, история навсегда.
- **Стабильные превью** — OG зашивается в момент создания, не зависит от того, что сделает источник через год.
- **Развёртывается за вечер** для тех, у кого есть домен и аккаунты GitHub + Cloudflare + Telegram.

## Архитектура одной строкой

`Mini App → Cloudflare Worker → GitHub Trees API → Pages rebuild → /l/{slug}.html`

Подробнее: см. [`ARCHITECTURE.md`](./ARCHITECTURE.md).

## Старт у себя

Два пути:

- **A. Локальный CLI** — `bash scripts/bootstrap.sh`, нужны `gh` и `wrangler` локально.
- **B. GitHub Actions wizard** — Use this template → Actions → Run workflow `bootstrap`, всё с мобилы.

См. [`docs/BOOTSTRAP.md`](./docs/BOOTSTRAP.md).

## Стек

- **Хостинг страниц:** GitHub Pages (бесплатно).
- **Логика:** Cloudflare Workers (free tier с запасом).
- **Хранилище:** репозиторий — `index.json` (журнал) + `/l/*.html` (рабочие файлы).
- **Кэш OG/идемпотентность:** Workers KV (опционально, для гибрида).
- **Mini App:** ванильный HTML + JS, адаптер платформы.
