# Конспект инвариантов

Документ для быстрой сверки в любой сессии Code. Если что-то противоречит этому списку — это баг (либо в коде, либо в этом списке; обсуждаем перед изменением).

## Архитектурные

1. **Источник правды — git.** `index.json` + `/l/*.html`. KV — только кэш или гибридный буфер, никогда не основное хранилище.
2. **Одна ссылка — один файл `/l/{slug}.html`.** Нет общего роутера, нет 404-трюков, нет SPA на корневой странице. GH Pages отдаёт страницу напрямую с кодом 200.
3. **Одна пачка (1–5 URL) — один коммит.** Через GitHub Trees API, не через серию `PUT /contents`.
4. **OG зашивается в момент создания.** Авто-обновления нет. Только команда `/refresh {slug}` или кнопка в админке.
5. **HTML шаблон содержит UA-sniffing.** Боты превью видят OG без редиректа. Люди — редирект после отстрела Метрики.

## Поведенческие

6. **Slug проверяется на коллизии всегда.** В режиме `counter` это страховка от рассинхрона, в `random` — обязательная часть алгоритма.
7. **Slug-конфликт внутри одного батча невозможен.** Генератор знает про уже сгенерированные в текущем запросе.
8. **Авторизация всегда двухступенчатая:** подпись initData/launchParams + allowlist `userId`. Одно без другого недостаточно.

## Платформенные

9. **Ядро не знает про мессенджер.** Mini App через адаптер `PlatformAdapter`. Worker через два слоя верификации: общий `verifyAuth(platform, raw)`, специфичный — внутри адаптера платформы.
10. **TG — основной, VK и MAX — задел.** Скелеты адаптеров есть, реализация TG полная, VK/MAX содержат TODO с указанием на доки платформы.

## Развёртывание

11. **Bootstrap идемпотентен.** Каждая из 8 стадий безопасна к перезапуску.
12. **Секреты только в GitHub Secrets и переменных окружения Worker.** Никогда не в репо.
13. **Шаблон `wrangler.toml.tmpl` хранится в репо.** Развёрнутый `wrangler.toml` — нет (gitignore).

## Лицензия и атрибуция

14. **MIT** — для удобства форков.
15. **Branding/UI кастомизируется через `config/brand.json`.** Не редактированием шаблонов. Это позволяет форкам обновляться без конфликтов.
